# Ruboty::Wei
Ruboty plugin for responding with ｳｪｲ.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-wei', github: 'mzp/ruboty-wei'
```

And then execute:

    $ bundle

## Usage

```
> ruboty ｳｪｲ
ｳｪｲw
```

## Env

 * `WEI_ALL`: all: respond with all ｳｪｲ

```
> ｳｪｲ
ｳｪｲw
```
